# محرر ماركداون النخبوي (Elite Markdown Editor)

محرر ماركداون احترافي ومتطور يعمل بالكامل في المتصفح ويدعم العمل بدون اتصال بالإنترنت (Offline PWA). تم تصميم الأداة لتوفير بيئة كتابة خالية من التشتت، مع معاينة حية سريعة ودعم كامل للغة العربية.

## ✨ المميزات الرئيسية

* **العمل بدون إنترنت (PWA):** يمكنك تثبيت المحرر كتطبيق على حاسوبك أو هاتفك واستخدامه في أي وقت بدون الحاجة للإنترنت.
* **البحث المتقدم:** محرك بحث مدمج يدعم تجاهل التشكيل العربي (الفتحة، الضمة، الكسرة، إلخ) للوصول للنتائج بدقة.
* **إدارة الملفات المتقدمة:** 
  * تكامل مع `File System Access API` لفتح الملفات وتعديلها وحفظها مباشرة على جهازك (حفظ سريع).
  * خيارات تصدير متعددة (Markdown, HTML, Text).
  * طباعة احترافية بصيغة PDF مع الحفاظ على تنسيق الأكواد والألوان.
* **تاريخ التعديلات (Undo/Redo):** نظام تراجع متقدم يحفظ حالة النص واسم الملف المرتبط به.
* **المعاينة الحية:** معاينة التنسيقات بشكل فوري بالتزامن مع الكتابة.
* **دعم الأكواد والرياضيات:**
  * تلوين احترافي للأكواد البرمجية البرمجية (Syntax Highlighting) عبر `Highlight.js`.
  * دعم كتابة المعادلات الرياضية المعقدة عبر `KaTeX`.
* **تصميم مرن ومتجاوب:** واجهة مستخدم (UI) مظلمة وحديثة (Dark Mode) تتكيف مع شاشات الهواتف والحواسيب.

## 🛠 التقنيات المستخدمة

* **HTML5 / CSS3 / Vanilla JS** - بدون إطارات عمل ثقيلة لضمان السرعة.
* [Marked.js](https://marked.js.org/) - لتحويل الـ Markdown إلى HTML.
* [DOMPurify](https://github.com/cure53/DOMPurify) - لفلترة وتنظيف الكود (XSS Protection).
* [Highlight.js](https://highlightjs.org/) - لتلوين الأكواد.
* [KaTeX](https://katex.org/) - لتصيير المعادلات الرياضية بشكل احترافي.

## 🚀 كيفية الاستخدام والتثبيت

1. قم بفتح التطبيق من خلال الرابط (أو عبر استضافته محلياً).
2. اضغط على أيقونة **"تثبيت"** في شريط المتصفح لتنصيب التطبيق كـ PWA على جهازك.
3. يمكنك البدء في الكتابة فوراً أو فتح ملف `.md` أو `.txt` من جهازك.
4. لحفظ تقدمك بشكل سريع لنفس الملف، استخدم زر **"حفظ سريع"**، ولحفظه كملف منفصل استخدم زر **"تصدير"**.

## 💡 الاختصارات

* `Ctrl + O`: فتح ملف
* `Ctrl + S`: حفظ سريع للملف المفتوح
* `Ctrl + Z`: تراجع
* `Ctrl + Y`: إعادة
* `Ctrl + P`: طباعة / تصدير PDF
